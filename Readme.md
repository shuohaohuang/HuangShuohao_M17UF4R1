# Controles del Juego 🎮

📌 **Movimiento**

- **W** → Mover hacia adelante
- **A** → Mover a la izquierda
- **S** → Mover hacia atrás
- **D** → Mover a la derecha

⚔️ **Ataque**

- **Espacio** → Atacar a todos los enemigos

# 🛠️ Debug de Raycast

📌 **Colores en Debug.DrawRay**

- 🟡 **Amarillo** → El objetivo **ha sido detectado** y el enemigo **lo está persiguiendo**.
